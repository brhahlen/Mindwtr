# Desktop Keybindings

Mindwtr Desktop supports two keyboard shortcut presets: **Vim** and **Emacs**.  
Change presets in **Settings → Keyboard Shortcuts**. Press `?` (Vim) or `Ctrl-h` (Emacs) anytime to open the in‑app help.

## Vim Preset (Default)

**Global**
- `/` Open global search
- `?` Show keyboard shortcuts
- `g` + key to navigate:
  - `gi` Inbox
  - `gn` Next Actions
  - `ga` Agenda
  - `gp` Projects
  - `gc` Contexts
  - `gr` Review
  - `gw` Waiting For
  - `gs` Someday/Maybe
  - `gl` Calendar
  - `gb` Board
  - `gd` Done
  - `gA` Archived

**Task List (when a task list is visible)**
- `j` / `k` Move selection down/up
- `gg` / `G` Jump to top/bottom
- `e` Edit selected task
- `x` Toggle done for selected task
- `dd` Delete selected task
- `o` Focus add‑task input

## Emacs Preset

**Global**
- `Ctrl-s` Open global search
- `Ctrl-h` Show keyboard shortcuts
- `Alt` + key to navigate:
  - `Alt-i` Inbox
  - `Alt-n` Next Actions
  - `Alt-a` Agenda
  - `Alt-p` Projects
  - `Alt-c` Contexts
  - `Alt-r` Review
  - `Alt-w` Waiting For
  - `Alt-s` Someday/Maybe
  - `Alt-l` Calendar
  - `Alt-b` Board
  - `Alt-d` Done
  - `Alt-A` Archived

**Task List**
- `Ctrl-n` / `Ctrl-p` Move selection down/up
- `Ctrl-e` Edit selected task
- `Ctrl-t` Toggle done
- `Ctrl-d` Delete selected task
- `Ctrl-o` Focus add‑task input

## Notes

- Shortcuts are ignored while typing in inputs/textareas and when the Inbox processing wizard is open.
- This is the Phase‑2 MVP shortcut set; more actions will be added later (bulk actions, capture, etc.).
